## Instructions

To add or update team member images:

1. Add your image files to this `/public/team/` directory
2. Use descriptive names like `member-4.jpg`, `member-5.jpg`, etc.
3. Update the image paths in `/app/team/page.tsx` in the `coreTeam` array
4. Recommended image size: 400x400px or larger (square aspect ratio works best)
5. Supported formats: .jpg, .jpeg, .png, .webp
