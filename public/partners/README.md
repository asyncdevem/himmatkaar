# Partner Logos

Place partner logo images in this directory.

## Recommended Specifications:
- Format: PNG with transparent background
- Size: 200x200px (or similar square/rectangular ratio)
- File naming: lowercase with hyphens (e.g., `unicef.png`, `red-cross.png`)

## How to Add:
1. Save your logo file in this directory
2. Update the partner name in `app/page.tsx` if needed
3. The logo will automatically display in the partner network section
